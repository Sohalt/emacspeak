12.55 emacspeak-url-template Options

Commentary: It is often useful to have “parametrized hot list entries” i.e., hotlist entries that are “templates” for the actual URL. The user provides values for the parametrized portions of the URL e.g. the date. See See URL Templates, for details on the URL templates that are presently defined.

Automatically generated documentation for options defined in module emacspeak-url-template. These options are customizable via Emacs' Custom interface.

— Variable: User Option emacspeak-url-template-google-calendar-uri

URI for accessing mobile version of Google Calendar. Google Apps users should set this to http://www.google.com/calendar/a/<my-corp>/m?output=xhtml

— Variable: User Option emacspeak-url-template-reading-list-opml

OPML feed location to use for our Custom Search.

— Variable: User Option emacspeak-url-template-weather-city-state

Default city/state for weather forecasts
