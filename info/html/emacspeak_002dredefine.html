No Commentary

Automatically generated documentation for commands defined in module emacspeak-redefine.

— Interactive Command: emacspeak-backward-char nil

Not bound to any key.

Backward-char redefined to speak char moved to.

— Interactive Command: emacspeak-forward-char nil

Not bound to any key.

— Interactive Command: emacspeak-self-insert-command nil

Not bound to any key.

Insert a character. Speaks the character if emacspeak-character-echo is true. See command emacspeak-toggle-word-echo bound to C-e d w. Speech flushes as you type.
