Automatically generated documentation for commands defined in module emacspeak-dismal.

— Interactive Command: emacspeak-dismal-backward-col-and-summarize nil

Not bound to any key.

Move backward by arg columns (the previous column by default)and summarize it.

— Interactive Command: emacspeak-dismal-backward-row-and-summarize nil

Not bound to any key.

Move backward by arg rows (the previous row by default)and summarize it.

— Interactive Command: emacspeak-dismal-col-summarize nil

Not bound to any key.

— Interactive Command: emacspeak-dismal-display-cell-expression nil

Not bound to any key.

— Interactive Command: emacspeak-dismal-display-cell-value nil

— Interactive Command: emacspeak-dismal-display-cell-with-col-header nil

Not bound to any key.

Display current cell along with its column header. The `column header' is the entry in row 0.

— Interactive Command: emacspeak-dismal-display-cell-with-row-header nil

Not bound to any key.
