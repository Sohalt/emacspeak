11.33 emacspeak-gnus

No Commentary

Automatically generated documentation for commands defined in module emacspeak-gnus.

— Interactive Command: emacspeak-gnus-summary-catchup-quietly-and-exit nil

Not bound to any key.

Catch up on all articles in current group.

— Interactive Command: emacspeak-gnus-summary-hide-all-headers nil

Not bound to any key.

Hide all headers in the article. Use this command if you don't want to listen to any article headers when reading news.

— Interactive Command: emacspeak-gnus-summary-show-some-headers nil

Not bound to any key.

Show only the important article headers, i.e. sender name, and subject.
