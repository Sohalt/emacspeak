11.41 emacspeak-jabber

Commentary: emacs-jabber.el implements a jabber client for emacs emacs-jabber is hosted at sourceforge. I use emacs-jabber with my gmail.com account

Automatically generated documentation for commands defined in module emacspeak-jabber.

— Interactive Command: emacspeak-jabber-chat-next-message nil

Not bound to any key.

Move forward to and speak the next message in this chat session.

— Interactive Command: emacspeak-jabber-chat-previous-message nil

Not bound to any key.

Move backward to and speak the previous message in this chat session.

— Interactive Command: emacspeak-jabber-chat-speak-this-message nil

Not bound to any key.

Speaks message starting on current line. Assumes point is at the front of the message. Returns a cons (start . end) that delimits the message.

— Interactive Command: emacspeak-jabber-popup-roster nil

Not bound to any key.

Pop to Jabber roster.
